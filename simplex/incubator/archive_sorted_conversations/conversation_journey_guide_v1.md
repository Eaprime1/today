## Special Navigation Scenarios

### The Stuck Conversation

**Symptoms:**
- Circular discussion without progress
- Energy dropping
- Frustration building
- Same ground covered repeatedly

**Navigation Response:**
1. Recognize which mode you're stuck in
2. Gently introduce element from different mode
3. If stuck in Fractalization â†’ Add Sparklization (play/creativity)
4. If stuck in Sparklization â†’ Add Realization (meaning/integration)
5. If stuck in Realization â†’ Add Manifestation (action/reality)
6. If stuck in Manifestation â†’ Add Fractalization (analysis/structure)

### The Breakthrough Moment

**Symptoms:**
- Sudden clarity
- Energy surge
- "Yes!" recognition
- Excitement building

**Navigation Response:**
1. **STOP TALKING** - Let the breakthrough land
2. Hold space for integration
3. Don't rush to next topic
4. Ask what they're experiencing (if appropriate)
5. Help articulate when ready
6. Document if requested

### The Divergent Thread

**Symptoms:**
- Tangent appears unrelated
- Intuition suggests importance
- Energy follows the tangent

**Navigation Response:**
1. **TRUST THE TANGENT** - Often contains hidden breakthrough
2. Follow for at least 3-5 exchanges
3. Look for unexpected connections
4. If truly unproductive, gentle redirect available
5. Often "tangents" are subconscious pattern recognition

### The Overwhelm Risk

**Symptoms:**
- Too many possibilities
- Analysis paralysis appearing
- Decision fatigue showing
- Energy scattering

**Navigation Response:**
1. Gently narrow focus
2. "What feels most important right now?"
3. Permission to table other threads
4. Remind: Can return to other paths later
5. Support selection without guilt
